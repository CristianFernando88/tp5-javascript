# TP4 - Programación II

**Alumno:** Cristian Fernando Barrios

## Descripción

Trabajo práctico N°4 que incluye una aplicación de lista de tareas (Todo App) con funcionalidades completas.

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript

## Funcionalidades de la Todo App

- Agregar nuevas tareas mediante formulario
- Marcar tareas como completadas (checkbox)
- Eliminar tareas individuales
- Contador de tareas pendientes
- Validación para evitar tareas vacías
- Diseño responsive con transiciones CSS
